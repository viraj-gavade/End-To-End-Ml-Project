About the Project

Forest Fire Weather Index Prediction

An end-to-end machine learning solution for predicting forest fire risk.

Project Overview

This project aims to predict the Forest Fire Weather Index (FWI) using machine learning techniques. The FWI is a metric used by fire management agencies to estimate fire danger and potential fire behavior.

The Dataset

We use the Algerian Forest Fires dataset, which contains meteorological data collected from two regions in Algeria: Bejaia region and Sidi-Bel Abbes region. The data spans the period from June 2012 to September 2012.

Features Used

  • Temperature: Day temperature in degrees Celsius
  • RH: Relative humidity in percentage
  • Ws: Wind speed in km/h
  • Rain: Outside rain in mm
  • FFMC: Fine Fuel Moisture Code index from the FWI system
  • DMC: Duff Moisture Code index from the FWI system
  • ISI: Initial Spread Index from the FWI system
  • Classes: 0 for not fire, 1 for fire
  • Region: 0 for Bejaia region, 1 for Sidi-Bel Abbes region

Model Used

We use a Ridge Regression model, which is a linear regression technique that includes L2 regularization. Ridge regression helps prevent overfitting and provides more stable predictions.

Technology Stack

  • Python for data processing and modeling
  • Scikit-learn for machine learning algorithms
  • Flask for web application development
  • Tailwind CSS for modern UI design
  • JavaScript for enhanced user interactions

How It Works

Users input various meteorological parameters into the form. The application then processes this data, scales it appropriately using a pre-trained StandardScaler, and feeds it into the Ridge Regression model. The model computes the predicted FWI value, which is then displayed to the user along with an interpretation of the risk level.
